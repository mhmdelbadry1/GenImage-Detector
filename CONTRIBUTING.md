# Contributing to GenImage-Detector

Thank you for your interest in contributing to GenImage-Detector! This document provides guidelines for contributing to the project.

## How to Get Involved

### 1. No Need to Ask for Username!

Instead of asking the repository owner for their username, you can:

- **Star the Repository**: Show your interest by starring https://github.com/mhmdelbadry1/GenImage-Detector
- **Fork and Contribute**: Fork the repository and submit pull requests
- **Request Access**: Use our [Access Request Template](https://github.com/mhmdelbadry1/GenImage-Detector/issues/new?template=request_access.md) to become a collaborator

### 3. Getting Started

1. **Fork the repository** to your GitHub account
2. **Clone your fork** locally:
   ```bash
   git clone https://github.com/YOUR-USERNAME/GenImage-Detector.git
   cd GenImage-Detector
   ```
3. **Create a branch** for your changes:
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Make your changes** and commit them:
   ```bash
   git add .
   git commit -m "Description of your changes"
   ```
5. **Push to your fork**:
   ```bash
   git push origin feature/your-feature-name
   ```
6. **Open a Pull Request** on GitHub

### 4. Pull Request Guidelines

- Write clear, descriptive commit messages
- Include tests for new features
- Update documentation as needed
- Follow the existing code style
- Keep pull requests focused on a single feature or fix

### 6. Reporting Issues

When reporting issues, please include:
- A clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Your environment (OS, Python version, etc.)
- Screenshots if applicable

### 7. Feature Requests

We welcome feature requests! Please:
- Check if the feature has already been requested
- Clearly describe the feature and its benefits
- Provide examples of how it would be used

### 8. Becoming a Collaborator

If you're interested in becoming a direct collaborator with write access:

1. **Option A**: Open an [Access Request Issue](https://github.com/mhmdelbadry1/GenImage-Detector/issues/new?template=request_access.md)
2. **Option B**: Make several quality contributions via pull requests
3. **Option C**: Contact the repository owner with your GitHub username

The owner will review your request and send you an invitation if approved.

### 9. Community Guidelines

- Be respectful and inclusive
- Provide constructive feedback
- Help others when you can
- Follow GitHub's [Community Guidelines](https://docs.github.com/en/site-policy/github-terms/github-community-guidelines)

### 10. Questions?

If you have questions:
- Open a [Discussion](https://github.com/mhmdelbadry1/GenImage-Detector/discussions) (if enabled)
- Open an [Issue](https://github.com/mhmdelbadry1/GenImage-Detector/issues)
- Contact the maintainer
